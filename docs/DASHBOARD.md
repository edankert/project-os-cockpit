# Project Dashboard

## Cockpit Layout

This project uses a three-pane cockpit layout in Obsidian:
- **Left sidebar**: Pin `__bases__/NAV.base` — browse features, phases, and issues
- **Center**: Editor — click any item to open it
- **Right sidebar**: Pin `__bases__/CONTEXT.base` — automatically shows tasks, requirements, issues, and tests related to the active note

See [[OBSIDIAN]] for workspace setup instructions.
