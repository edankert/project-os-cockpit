# `tools/` (Docs tooling)

This directory holds supporting material for the documentation system (instructions, conventions, and agent/tool collateral).

## What goes here
- Normative instructions under `tools/instructions/`.
- Optional agent/tool collateral under `tools/agents/` and `tools/skills/`.

## Project entrypoints (real repo)
- Record your project’s real “front door” scripts/commands in workflow notes under `../docs/workflows/`.

## What does not go here
- Project documentation content (belongs under `../docs/`).

## Index
- Instructions: `instructions/README.md`
- Agents: `agents/README.md`
- Skills: `skills/README.md`
